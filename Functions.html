JavaScript Higher Order Functions

Array methods:
filter(SCRIPTS, script => script.living) this builds a new array of the elements that passed the test which is script => script.living.
map(rtlScripts, s => s.name) === rtlScripts.map(s => s.name) this returns a new array with the same length which is transformed by the function passed in after the array.
reduce([1, 2, 3, 4], (a, b) => a + b, 0) this takes (an array, a function, a start index) and manipulates the array acording to the function this example adds together the array
